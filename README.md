# Escape from Duckov - Haptic Feedback Mod

Controller vibration/haptic feedback for Escape from Duckov. Feel every shot!

## Features

- **Weapon-specific vibration patterns**: Different feel for pistol, SMG, rifle, shotgun, sniper, melee, and heavy weapons
- **Kill feedback**: Vibration when you eliminate enemies
- **Headshot feedback**: Stronger vibration for critical hits
- **In-game settings**: Configure via ModConfig (if installed)
- **Steam Deck optimized**: Works great on Steam Deck
- **Xbox/XInput support**: Works with Xbox and XInput-compatible controllers

## Requirements

- **Steam Deck** or **XInput-compatible controller**
- **Escape from Duckov** (Steam version)
- **Optional**: ModConfig mod for in-game settings

## Installation

### Steam Workshop
1. Subscribe to the mod on [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3637049322)
2. Launch the game
3. Enable the mod in the Mods menu

## Configuration

If you have **ModConfig** installed, settings appear in the mod options menu:

| Setting | Description | Default |
|---------|-------------|---------|
| Enabled | Enable/disable haptic feedback | On |
| Fire Vibration Intensity | Strength of shooting feedback | 0.6 |
| Fire Vibration Duration | How long fire vibration lasts (ms) | 100 |
| Kill Vibration Intensity | Strength of kill feedback | 0.9 |
| Headshot Vibration Intensity | Strength of headshot feedback | 1.0 |
| Damage Vibration Intensity | Strength when taking damage | 0.8 |
| Damage Vibration Duration | How long damage vibration lasts (ms) | 200 |

Press **F9** in-game to open the debug window for testing vibration.

## Vibration Patterns

| Weapon Type | Feel |
|-------------|------|
| Pistol | Light, quick pulse |
| SMG | Rapid, moderate |
| Rifle | Medium punch |
| Shotgun | Heavy thump |
| Sniper | Maximum recoil feel |
| Melee | Sharp impact |

## Troubleshooting

### No vibration
1. Ensure controller is connected before launching game
2. Check controller works in other games
3. Press F9 to open debug window and test vibration buttons
4. On Steam Deck, ensure you're using the built-in controls

### Vibration feels wrong
- Adjust intensity settings via ModConfig
- Try the test buttons in the F9 debug window
